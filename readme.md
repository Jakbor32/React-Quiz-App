Simple quiz app made for study. Project built with react using vite and react hook form.

## This project was bootstrapped with Vite.

https://vitejs.dev/

## How to run?

`npm run dev`

## Extra package npm form

https://react-hook-form.com/

`npm i react-hook-form`
